Getting Started

Learn how to use TraderPlus Editor efficiently

Thank you for using TraderPlus Editor! This application provides powerful tools to configure and manage your TraderPlus Configuration for DayZ servers using the TraderPlus v2.

add_circle

Create Default Config

Start fresh with a default configuration. This option creates a standard set of trader config files that you can customize to your needs.

import_export

Import Existing Config

Already have TraderPlus config files? Import them directly into the editor to modify your existing setup without starting over.

auto_fix_high

Convert from Other Mods

Migrate from other trader mods to TraderPlus v2. Our converter tool supports various format conversions to help you transition.
