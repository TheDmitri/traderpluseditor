Getting Started

Learn how to use TraderX Editor efficiently

Thank you for using TraderX Editor! This application provides powerful tools to configure and manage your TraderX Configuration for DayZ servers using the TraderX.

add_circle

Create Default Config

Start fresh with a default configuration. This option creates a standard set of trader config files that you can customize to your needs.

import_export

Import Existing Config

Already have TraderX config files? Import them directly into the editor to modify your existing setup without starting over.

auto_fix_high

Convert from Other Mods

Migrate from other trader mods to TraderX. Our converter tool supports various format conversions to help you transition.
